*Основные методы*

- GET - запрашивает представление ресурса (без изменения).
- HEAD - запрашивает ресурс аналогично GET, но без тела ответа.
- POST - используется для отправки сущностей к определённому ресурсу / отправка данных на сервер. Может изменять состояния.
- PUT - полная замена ресурса данными запроса.
- DELETE - удаляет указанный ресурс.
- OPTIONS - выводит, какие методы/параметры поддерживает эндпоинт.
